Map Pathfinding

Visualize shortest paths using A* and Dijkstra's algorithms

Controls

How to Use

  1. Upload a map image and corresponding CSV file with valid path points
  2. Click on the map to set your start point (marked with a green marker)
  3. Click again to set your end point (marked with a red marker)
  4. Choose your preferred pathfinding algorithm
  5. Click "Find Path" to visualize the shortest route
